Mi experiencia

A lo largo de los años he ido adquiriendo la siguiente experiencia y he ultilizado las siguientes teconologías:

HTML

HTML (Hypertext Markup Language) es el lenguaje estándar utilizado para crear y estructurar páginas web. Consiste en una serie de elementos y etiquetas que indican al navegador cómo mostrar el contenido, como texto, imágenes, enlaces y otros recursos.

CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento HTML. Permite controlar el diseño, los colores, las fuentes y el formato de una página web de manera independiente de su contenido.

Info .... .... ....

JS

JavaScript (JS) es un lenguaje de programación utilizado para crear contenido interactivo y dinámico en páginas web. Permite a los desarrolladores manipular el HTML y CSS de una página después de que haya sido cargada.
